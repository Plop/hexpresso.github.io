[ NEWS ][15/05/2015]

Hexpresso is now on Github !

We have decided to migrate on Github for many reasons:

  • Wordpress is not really pleasant to use,
  • Github is more simple to order our writeups by CTF,
  • Github allows us to create and modify very simply our WU, by merging using git,
  • People reading our WU will be able to clone our repo which might contains binaries, pictures, etc.
    • So now, we will migrate slowly all our WU on Wordpress to our Github. We hope you will like it :)

[ Hexpresso story and composition ]

We are Hexpresso, a French security group. Hexpresso was created in February 2013 and currently has 10 active members. Quite often, we participate in both online and offline security Capture The Flag competitions. We often publish after some of them Writeups on CTF tasks that you can find on our github.
